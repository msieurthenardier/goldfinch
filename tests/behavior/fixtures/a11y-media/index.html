Accessibility media fixture

A page carrying one image, one audio clip, and one video so Goldfinch's media panel catalogs an image / audio / video card (and the docked player, media-pick checkboxes, and image lightbox render) for the npm run a11y axe sweep. Assets are same-directory local files referenced over HTTP — the media panel never catalogs file:// sources, and there is deliberately no <iframe> embed (only youtube/vimeo-style embeds are cataloged).
